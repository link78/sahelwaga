# Sahelwaga

A lightweight, browser-based task manager built with vanilla HTML, CSS, and JavaScript.

## Features

- Add, complete, and delete tasks
- Filter tasks by **All**, **Active**, or **Completed**
- Clear all completed tasks at once
- Tasks are persisted in `localStorage` — they survive page refreshes

## Usage

Open `index.html` in any modern web browser — no build step or server required.

```
open index.html
```

## Project structure

```
sahelwaga/
├── index.html   # App shell & markup

└── app.js       # Application logic
```
